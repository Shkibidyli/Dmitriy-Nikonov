# Приложение "Weather Diary" (Дневник погоды)
### Никонов Дмитрий

### Описание проекта
Данное приложение представляет собой электронный дневник для фиксации погодных условий. Оно позволяет пользователю записывать наблюдения, сохранять их в структурированном виде и находить нужные данные с помощью встроенных фильтров.

### Основной функционал
*   **Добавление записей:** Ввод даты, температуры, текстового описания и отметки об осадках.
*   **Валидация:** Проверка на корректность ввода числовых данных.
*   **Хранение данных:** Автоматическое сохранение и загрузка истории из файла `weather.json`.
*   **Фильтрация:** Поиск записей по конкретной дате или просмотр дней, где температура выше заданного порога.

### Технологии
*   **Язык:** Python 3.x
*   **Библиотека GUI:** Tkinter
*   **Формат данных:** JSON

### Как запустить приложение
1. Убедитесь, что у вас установлен Python.
2. Скачайте файлы проекта или склонируйте репозиторий
3. Перейдите в папку проекта.
4. Запустите основной скрипт:
   `python main.py` (или `python3 main.py`)

### Примеры использования (Тесты)
1. **Тест на ввод данных:** Попробуйте оставить поле "Температура" пустым или ввести текст — программа выдаст предупреждение.
2. **Тест фильтра:** Добавьте записи за разные даты (например, 2026-10-01 и 2026-10-02). Введите первую дату в поле фильтра и нажмите "Поиск" — в таблице останется только одна запись.
